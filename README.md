## Project Overview

HRMS Lite is a lightweight Human Resource Management System built using Django and Django REST Framework.  

The application allows an admin to:

- Manage employee records
- Track daily attendance
- View attendance summary on a dashboard

This project demonstrates end-to-end full-stack development including:

- Backend API development
- Database modeling
- Server-side validation
- Template-based frontend
- Production deployment on Render
- PostgreSQL integration
